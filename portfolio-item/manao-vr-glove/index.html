SenseGlove Manao VR Glove

2020

Project Type: Master's Program Group Project

Client: SenseGlove

Personal Role: electronics and interactive systems, 3D rendering and animation, 3D printing technician

Re-imagining the SenseGlove as a mobile, soft glove for improved immersion into the VR Environment

Overview

The diagram below showcases the 3 development areas of the project. My main contribution in the project were in developing the finger tracking technology and interactive system used in the final prototype design.

Electronics and Interactive Systems

For the final prototype showcase, I used the integrated flex sensors with Arduino to control the virtual hand in the interactive environment I created using Python and Blender 3D. This allowed us to test the final glove prototype with potential users and obtain a real-time simulation of their hand movement on the computer screen as it would be captured in a more complex VR simulation environment.
